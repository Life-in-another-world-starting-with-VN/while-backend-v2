게임 플래이

게임은 미연시 게임이며 게임 안에는 각각의 세션으로 나누어져 있고 각 세션안에는 씬으로 나눠져 있습니다. 각 세션과 씬은 LLM이 자동으로 알아서 생성하게끔하면 됩니다.

세션은 한 장소를 단위로 해서 이루어지는 한 장면입니다. 씬은 해당 세션안에 있는 각 대사 or 선택지가 됩니다.

배경생성은 각 세션별로 진행하면 됩니다.

세션의 갯수는 게임 처음 생성시 정해지게 되고 세션의 내용 및 씬의 갯수 내용은 동적으로 LLM과 사용자의 대화 or 선택에 따라 달라지게 됩니다.(달라지게 하는것도 LLM의 선택)

게임 관련 정보 : 캐릭터는 총 3명이 있으며 사전 DB에 저장되어 있다.(DB에는 캐릭터id, 이름, 특징 or 성격)

한 세션의 마지막 씬 생성 요청시 다음 세션의 내용(content) 선정

배경생성 AI 클래스 - 배경 생성시 사용하면 됨

from google import genai
import os

from dotenv import load_dotenv
import requests

load_dotenv()

class BackgroundGenerator:
    def __init__(self):
        self.__fal_key = os.getenv("FAL_KEY")
        self.__fal_url = os.getenv("FAL_URL")
        self.__immage_size = os.getenv("IMAGE_SIZE")

        GEMINI_API_KEY = os.getenv("GEMINI_TOKEN")
        self.__client = genai.Client(api_key=GEMINI_API_KEY)
        self.gemini_model = os.getenv("GEMINI_MODEL")

    def create_background_image(self, story:str) -> str:
        prompt = self.__set_prompt(story)
        background_search_keyword = self.__get_search_word(prompt)
        print(background_search_keyword)

        return self.create_background_image_by_keyword(background_search_keyword)

    def create_background_image_by_keyword(self, keyword:str) -> str:
        background_image = self.__create_background_image(keyword)

        return background_image

    def __set_prompt(self, story:str) -> str:
        return (
                "미연시 게임 배경 이미지를 생성할 영어 검색어가 필요합니다.\n"
                f"장면 설명: {story}\n\n"
                "이 장면의 장소, 날씨, 분위기를 정확히 반영한 영어 검색어를 만들어주세요.\n"
                "날씨가 중요한 경우 반드시 포함하세요 (rainy, sunny, cloudy, snowy 등).\n"
                "응답은 검색어만 영어로! 설명 금지!\n\n"
                "예시:\n"
                "- 소나기 → rainy school corridor\n"
                "- 교실 → sunny classroom\n"
                "- 밤 공원 → night park\n"
                "- 눈 내리는 거리 → snowy street\n\n"
                "검색어:"
        )

    def __get_search_word(self, prompt: str) -> str:
        response = self.__client.models.generate_content(
            model=self.gemini_model,
            contents=[prompt],
        )

        return response.candidates[0].content.parts[0].text

    def __create_background_image(self, background_search_keyword):
        image_style = """
        visual novel style,
        clean line art,
        Makoto Shinkai-inspired sky,
        studio-quality anime background,
        romantic visual novel scene,
        NO PEOPLE,
        NO CHARACTERS,
        environment only,
        empty scene
        """
        prompt = f"{background_search_keyword}, {image_style}, {self.__immage_size}, ultra detail, no people, environment background"

        payload = {"prompt": prompt, "aspect_ratio": f"{self.__immage_size}"}

        headers = {
            "Authorization": f"Key {self.__fal_key}",
            "Content-Type": "application/json"
        }

        res = requests.post(self.__fal_url, headers=headers, json=payload)
        res.raise_for_status()

        image_url = res.json()["images"][0]["url"]

        return image_url

씬 생성 및 세션 내용 생성 시 제미나이 AI 사용예정

새 게임 생성

HTTP 메서드 : POST

URL : /api/v2/game

설명

새 게임 생성 하기 api

캐릭터 성격, 게임 장르, 플레이 시간을 입력 받아 새 게임 틀을 생성합니다. 처음 게임을 생성하면 전체 게임의 세션 갯수를 생성합니다.

처음 게임 생성시 첫 세션 및 첫 씬은 동적으로 미리 생성

header : 엑세스 토큰

Request body

{
  "personality": "따뜻함",
  "genre": "로맨스",
  "playtime": 5
}

personality : 캐릭터 성격

genre : 게임 장르

playtime : 분 단위(플래이 게임 시간)

Response(200) 

{
	"game_id" : 3,
	"personality" : "따뜻함",
	"genre" : "로맨스",
	"title" : "달달한 학교 로맨스",
	"playtime" : 5,
	"sesstions" : [
		{
			"sesstion_id" : 1,
			"content" : "학교 복도에서 1번 캐릭터를 만나 수학 수행평가 이야기를 나눈다."
			"scenes" : [
				{
					"role" : "아리아나",
					"scene_id" : 1,
					"type" : "dialogue",
					"dialogue" : "나 이번 수행평가 완전 망했어 ㅠㅠ",
					"selections" : []
				}
			],
			"background_url" : "https://....."
		}
	]
}

type에는 dialogue, selection 두가지가 있으며 dialogue라면 아래 dialogue 필드에 그냥 대사를 입력하면 되고 selection이라면 아래 selections 필드 안에 

1: 수행평가를 도와준다.
2: 그냥 쓴 웃음을 짓는다.

이런식으로 선택지를 넣어주면 된다. 선택지의 갯수 제한은 없다.

role은 현재 이 씬에서 어떤 사람이 말을 하는건지를 뜻한다. 그래서 아리아나면 아리아나 캐릭터가 유저에게 말을 하는것이다. 만약 type이 selection이라면 해당 씬에서는 유저가 선택을 해야 하는 씬이니까 role이 “user” 이런식으로 작성이 될것이다.
